What is Shell Scripting?

A shell script is a text file containing a series of commands that the shell interpreter (such as bash in Linux) executes one by one. Shell scripts automate repetitive tasks, execute programs, manipulate files, and manage system operations. The shell itself is a command-line interface between the user and the underlying operating system.

Writing Your First Shell Script
1. Open any text editor (e.g., nano, vi, gedit).
2. Start the script with a shebang line to specify the interpreter; for bash, use: #!/bin/bash
3. Write the commands you want the script to execute, each on a new line.
4. Save the file with a .sh extension, e.g. myscript.sh
5. Grant execute permission using: chmod +x myscript.sh
6. Run the script using: ./myscript.sh or bash myscript.sh

Basic Features and Syntax

• Comments: Start a comment with #. Anything after # on that line is ignored.
• Printing Output: Use echo to display text or variables.
→ echo "Hello, World!"
• Variables: Assign with = (no spaces) and use with $ prefix.
→ NAME="MOS Student" echo $NAME
• User Input: read command collects input from the user.
→ read NAME echo "You entered $NAME"
Conditional Statements: bash if [ $A -gt $B ] then echo "A is greater" else echo "B is greater" fi
• Loops:
For Loop: bash
for i in 1 2 3; do
echo $i
done
While Loop:
bash
i=1
while [ $i -le 5 ]; do
echo $i
i=$((i+1))
done
• Useful Shell Commands:
→ pwd – Print working directory
→ ls – List directory contents
→ cd – Change directory
→ cp, mv, rm – File copy, move, remove
→ man – Show manual for a command
→ chmod – Change file permissions (e.g., make script executable)
• Passing Arguments:
Access command-line arguments using $1, $2, ... for first, second, etc.
→ echo "First argument is $1"

Example: A Simple Script
#!/bin/bash
echo "Hello, MOS Student!"
echo "Today is: $(date)"
Save as welcome.sh, make executable, then run to see output.
